Intro into SCSS (SASS)

What

SCSS = preprocessor 4 CSS

Why

  • Nested Styles
  • $Variables
  • Mixins
  • Extends
  • Imports
  • = faster development

Another language to learn?

No

100% compatible with CSS

Nested Styles


			.myspecial .component .thing .btn {
				background-color: #000;
				color: #fff;
				border-radius: 5px;
				padding: 10px 20px;
			}

			.myspecial .component .thing .btn:hover {
				background-color: #fff;
				color: #000;
			}
			

				.myspecial .component .thing .btn {
					background-color: #000;
					color: #fff;
					border-radius: 5px;
					padding: 10px 20px;

					&:hover {
						background-color: #fff;
						color: #000;
					}
				}

Variables


			  $body: #226666;
			  $primary-color: #403075;
			  $footer: #AA8439;
			  $font-stack: Helvetica, sans-serif;
			  
	
					  body {
						  background: $body;
						  font: $font-stack;
					  }
  
					  .header {
						  color: $primary-color;
					  }
  
					  a {
						  color: $primary-color;
					  }

Mixins

	
				@mixin border-radius($round) {
					-webkit-border-radius: $round;
						-moz-border-radius: $round;
						-ms-border-radius: $round;
							border-radius: $round;
				}

				/*Just use '@include' directive to apply mixin */
				.box { @include border-radius(15px); }

Extends

	
				.box {
					margin: 10px;
					padding: 10px;  
					border: 2px solid blue;
				}
				  
				.box-red {
					@extend .box;
					border-color: red;
				}
				.box-yellow {
					@extend .box;
					border-color: yellow;
				}

Imports

	
				@import “variables”;
				@import “fonts”;
				@import “base”;
				@import “buttons”;
				@import “layout”;
